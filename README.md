# TipSplit — Tip Calculator & Bill Splitter

A simple single-screen tip calculator built with vanilla HTML, CSS, and JavaScript. Enter a bill amount, pick a tip percentage, and choose how many people are splitting — it updates live as you type.

## How to Run

No install needed. Just download or clone the repo and open `index.html` in your browser.

```
git clone https://github.com/Dev-Atta/tip-calculator.git
cd tip-calculator
```

Then open `index.html` directly in Chrome, Firefox, or any browser. That's it.

## Features

- Live calculations — no calculate button
- Tip presets (10%, 15%, 20%) plus a custom % input
- Splits the bill across any number of people
- Inline validation with error messages near each field
- Reset button to clear everything
- Works on mobile and desktop

## Built With

- HTML
- CSS (no frameworks)
- JavaScript (no libraries)
